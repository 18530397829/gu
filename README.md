# gu

Daily stock data and source code lists for the XTP-supported targets collected in this workspace.

## Data

- `akshare_xtp_daily_20260618/daily_csv/`: 174 daily CSV files.
- `akshare_xtp_daily_20260618/download_summary.json`: download summary.
- `akshare_xtp_daily_20260618/source_codes.csv`: source code list used for the download.

The daily data was downloaded with AkShare using unadjusted A-share daily bars.
